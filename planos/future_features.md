# Sugestões de Funcionalidades Futuras - Chatbot Seguros I2A2

Este documento lista possíveis evoluções para o sistema, organizadas por categoria.

## 1. Experiência do Utilizador (UX)
- **Upload de Fotos (OCR/Vision)**: Analisar danos em tempo real a partir de fotos enviadas.
- **Interface de Voz**: Integração de Speech-to-Text e Text-to-Speech.
- **Componentes Interativos**: Carrosséis para escolha de planos e formulários guiados para participação de sinistros.

## 2. Inteligência Artificial e RAG
- **Re-ranking Semântico**: Uso de Cross-Encoders para melhorar a precisão da recuperação.
- **Multi-document RAG**: Processamento de PDFs de Condições Gerais e Particulares.
- **Análise de Sentimento**: Detetar frustração e ajustar o tom do bot ou escalar para humano.

## 3. Integrações e Negócio
- **Consulta de Apólice (Backend)**: Integração com APIs core de seguros para verificação de dados reais.
- **WhatsApp Business**: Disponibilização do bot no canal WhatsApp.
- **Escalação para Humano**: Integração com sistemas de ticket (Zendesk/Intercom).

## 4. Analítica
- **Dashboard de Performance**: Visualização de métricas de acerto do RAG e dúvidas não resolvidas.
